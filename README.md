

# try
A short Python script to brute-force a command.

### Usage:
```sh
$ try git clone https://www.github.com/large/repository
's try - v1.0.0

Cloning into 'AAA' ...
... .... ... .... .. ..
```

### Installation:
`sudo make install` or `make install` as root.
